1.
Числа, кратные 3 или 5
Задача на самом деле достаточно простого уровня. Необходимо знать о существовании такой операции как: "Деление по модулю". В php эту операцию выполняет следующий оператор "%". Далее все просто, создаем цикл(For) в котором перебираем числа от 1 до 1000 (можно даже от 3, у нас ведь 1-й делитель равен 3). И в условии (If) при помощи оператора, указанного выше, мы проверяем целочисленное деление на "3" или соответственно "5", если одно из условий совпадает(Например 6/3==2, т.е. целое число), то это число (6) мы записываем в переменную sum. По итогу, перебрав все 1000 чисел в нашем цикле мы получаем итоговую сумму, в переменной sum. Ее и выводим, как решение. 
                    $sum=0;
                    for ($num=1;$num<1000; $num++)
                    {
                    if ($num%5==0 or $num%3==0) $sum+=$num;
                    }
                    echo 'Сумма элементов чисел массива кратных 3 или 5 составляет ', $sum, '<br>';
2.
Четные числа Фибоначчи
Ряд Фибоначчи представляет собой ряд чисел, каждое последующее из которых, представляет собой сумму 2-х предыдущих. Первые 2 числа заранее известны, это 0 и 1, следовательно 3-е число будет тоже 1, четвертое 2, далее 3,5,8,13 и тд. Определим сразу же первые 2 числа, ведь они нам 100% известны, это 0 и 1. Используя цикл, будем получать числа путем сложения 2-х предыдущих элементов. Главное после получения числа, определить следующие 2 элемента, теперь у нас первый элемент будет равняться предыдущему второму, а второй соответственно полученному числу Фибоначчи ну и тд. Цикл у нас бесконечный, поставим условие, что число Фибоначчи не должно превышать 4000000, при срабатывании условия, делаем выход и выводим результат на экран, в противном случае при помощи условия суммируем все элементы, проверяя их на кратность 2-м. 
                    $sum=0;
                    $FirstNumber=0;
                    $SecondNumber=1;
                    $Fib=1;
                    echo 'Формируем ряд Фибоначчи, элементы которого не превышают 4000000';
                    echo $FirstNumber;
                    echo $SecondNumber;
                    for ($i=1;;$i++){
                    $Fib=$FirstNumber+$SecondNumber;
                    $FirstNumber=$SecondNumber;
                    $SecondNumber=$Fib;
                    if ($Fib>4000000) {
                        break;
                    }else{
                        echo $Fib;
                        if ($Fib%2==0)$sum+=$Fib;
                    }
                    }
                    echo 'Сумма элементов кратных 2 = ', $sum;
